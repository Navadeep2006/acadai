## вљЎ Quick Start (Local Development)

### 1. Prerequisities
- Python 3.9+
- Node.js & npm

### 2. Backend Setup
```bash
# Install Python dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env and set SECRET_KEY at minimum
```

### 3. Frontend Setup
```bash
cd frontend-new
npm install
# For development with Hot Module Replacement:
npm run dev
```

### 4. Running the App (Development)
1. Run the Flask backend:
   ```bash
   python backend/app.py
   ```
2. Run the Vite frontend (if not already running):
   ```bash
   cd frontend-new && npm run dev
   ```
3. Open **http://localhost:5173** for the frontend or **http://localhost:5000** for the API.

## рџљЂ Production Build & Deployment

### Build the Frontend
Before deploying, you must build the frontend so Flask can serve it:
```bash
cd frontend-new
npm run build
```
The compiled files will be located in `frontend-new/dist/`.

### Deployment to Render.com
The project is configured for Render via `render.yaml`.
1. Connect your GitHub repository to Render.
2. Render will automatically detect `render.yaml` and set up the service.
3. The build command is: `cd frontend-new && npm install && npm run build && cd .. && pip install -r requirements.txt`.
4. The start command is: `gunicorn -b 0.0.0.0:$PORT backend.app:app`.

## рџ§  Machine Learning Engine
AcadAI uses a weighted ensemble of three models:
- **Random Forest (45%)**: Handles non-linear student behavior patterns.
- **Gradient Boosting (40%)**: Optimized for tabular performance data.
- **Ridge Regression (15%)**: Provides a stable linear baseline.

The model automatically retrains in the background when sufficient new data is added via Firebase.
